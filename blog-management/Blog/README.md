# Blog Management System

## 📌 Project Overview

This is a Blog Management System built using Flask, MongoDB, HTML, CSS,
and JavaScript.\
It allows users to perform CRUD (Create, Read, Update, Delete)
operations on blog posts.

## 🚀 Features

-   Add new blog posts
-   View all blog posts
-   Edit existing blog posts
-   Delete blog posts
-   Organized blog categories

## 🛠️ Tech Stack

-   Frontend: HTML, CSS, JavaScript
-   Backend: Flask
-   Database: MongoDB
